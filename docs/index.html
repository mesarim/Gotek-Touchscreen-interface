How to flash
  1. Select your device from the cards above.
  2. Connect your ESP32 to this computer via USB. Use a data cable, not a charge-only cable.
  3. If prompted, put the device in bootloader mode: hold BOOT, tap RESET, release BOOT.
  4. Click Install and select the correct COM port when asked.
  5. Wait for flashing to complete — typically 30–60 seconds.
  6. After flashing: insert a blank FAT32 SD card and power on. The device will auto-generate CONFIG.TXT and create the /ADF and /DSK folders.
⚠ XIAO only: Make sure the external WiFi antenna is plugged in before powering on. Without it, wireless range will be very poor.
⚠ All devices: USB CDC On Boot must be Disabled — this is already set in the pre-built firmware. If you rebuild from source, ensure this setting is correct or the Gotek will not recognise the device.

After flashing
  1. Place your .adf or .dsk files in the /ADF or /DSK folders on the SD card.
  2. Optionally add GameName.jpg (cover art) and GameName.nfo (info text) alongside each game.
  3. Connect the device to your Gotek via USB and power on your retro machine.
  4. Tap a game and press INSERT. Your retro machine sees a freshly loaded floppy.
  5. To manage games wirelessly: connect to the GotekWifi WiFi network and open http://192.168.4.1 (Waveshare 7" only).
