GOTEK TOUCHSCREEN INTERFACE

OMEGAWARE · JC3248 · interactive UI preview

Boot plays one of 6 cracktro styles (the device picks one per boot via CRACKTRO=; here they cycle) — tap to continue. Then: tap a game to select · tap again (or INSERT) to mount · A–Z strip jumps by letter · THEME cycles palettes · INFO holds FONT, ROTATE, COMPACT, and LOAD DIAG (mounts the built-in Amiga Test Kit — works with no SD card).
Faithful re-creation of the device UI for preview — not the firmware itself.
